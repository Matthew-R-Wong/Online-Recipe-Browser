Overview
    The Recipe Manager is a desktop application that allows you to add, view, edit, delete, and sort recipes using a graphical interface. Recipes include a name, photo reference, tags, ingredients, and a description. The interface updates automatically as changes are made.


Main Window Layout

The main window contains the following key elements:

Recipe Selector (Dropdown / Combobox)
    Used to choose which recipe is currently displayed.

Tag Filter List
    Allows you to select one or more tags to filter recipes.

Control Buttons
    Add Recipe
    Edit Recipe
    Delete Recipe
